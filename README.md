# Halma Game

To run simply just execute in terminal/cmd:
```
python halma_gui.py
```

Prerequisite: pygame library
Tested on pygame 1.9.6 (Python 3.7.6)

Font: Coolvetica (https://www.dafont.com/coolvetica.font)
